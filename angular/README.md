# Angular & Front-End Development

Repositório dedicado aos estudos e projetos desenvolvidos na disciplina de **Desenvolvimento Web Front-End**, focando na criação de interfaces modernas, reativas e performáticas.

### Tópicos em Foco:

* **Angular Signals:** Implementação do novo modelo de reatividade do Angular para gerenciamento de estado granular e otimização de performance.
* **Arquitetura de Componentes:** Criação de componentes reutilizáveis e modulares, seguindo as melhores práticas do ecossistema Angular.
* **Roteamento Avançado:** Configuração de rotas dinâmicas e proteção de acesso em aplicações Single Page (SPA).
* **TypeScript Moderno:** Uso de tipagem estrita, interfaces e decoradores para garantir a escalabilidade e segurança do código.
* **Consumo de APIs:** Integração com serviços externos e manipulação de dados assíncronos.

### Tecnologias e Ferramentas:

* **Framework:** Angular (v17+)
* **Linguagem:** TypeScript
* **Build Tool:** Angular CLI
